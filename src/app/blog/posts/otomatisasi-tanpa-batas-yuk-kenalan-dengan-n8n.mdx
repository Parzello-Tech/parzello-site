## âš¡ï¸ Apa Itu n8n?

**n8n** adalah _workflow automation tool_ yang revolusioner. Sifatnya **open source** dan **self-hosted**, yang berarti Anda memiliki kontrol penuh atas data Anda. n8n memungkinkan Anda menghubungkan lebih dari **500+ integrasi** (seperti Google Workspace, Telegram, OpenAI, dll.) untuk membuat alur kerja otomatis **tanpa perlu menulis kode**.

## âœ¨ Keunggulan n8n untuk Bekerja Cerdas

Mengapa n8n menjadi pilihan utama para _developer_ dan _creator_?

-   **Self-hosted & Keamanan Data:** Data Anda tetap aman di server pribadi Anda.
-   **Visual Editor:** Rancang alur kerja dengan mudah menggunakan _drag & drop node_.
-   **Extensible:** Anda dapat membuat _node_ kustom sendiri dengan JavaScript/TypeScript.
-   **Workflow Tak Terbatas:** Tidak ada batasan _task_ atau _trigger_ seperti pada kebanyakan layanan _cloud_.

## ğŸ§© Cara Kerja Dasar (Node)

Setiap _workflow_ n8n terdiri dari _Node_:

-   **Trigger Node:** Memulai alur kerja (Contoh: "Saat email baru masuk").
-   **Action Node:** Menjalankan tugas atau aksi (Contoh: "Kirim notifikasi ke Slack").

## ğŸš€ Quick Start (Self-Host)

Cara yang paling disarankan untuk mendapatkan kontrol penuh dan fungsionalitas tanpa batas adalah dengan _self-host_ menggunakan Docker. Berikut cara memulainya:

### 1. Install n8n menggunakan Docker

```bash
docker run -it --rm \
    --name n8n \
    -p 5678:5678 \
    -v ~/.n8n:/home/node/.n8n \
    n8nio/n8n
```

### 2\. Akses n8n

Setelah perintah Docker dieksekusi, buka browser Anda:

```bash
http://localhost:5678
```

### 3\. Alternatif Cepat (Cloud)

Jika Anda ingin langsung mencoba tanpa instalasi:

-   Kunjungi **n8n.io** untuk menggunakan layanan Cloud mereka.
